# Recomendation system of Christmas movies
Проект с рекомендациями фильмов и метриками похожести.

## Навигация по проекту

- `main.py` — основной скрипт проекта.
- `levenshtein.py` — функции расчета расстояния Левенштейна.
- `synthetic-creation.py` — генерация синтетических данных.
- `christmas_movies.csv` — исходный датасет фильмов.
- `christmas_movies_clean.csv` — очищенный датасет.

## Быстрый старт

```bash
python main.py
```
